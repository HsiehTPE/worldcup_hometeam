## 第四部分：计分说明

### 用户作答方式
- 每题从 A / B / C / D 中选择一个最符合自己的选项。

### 结果码生成规则
- Q5、Q8、Q9、Q15 统计一次，A / B / C / D 哪个最多，就得到第 1 位结果码。
- Q2、Q3、Q4、Q7 统计一次，A / B / C / D 哪个最多，就得到第 2 位结果码。
- Q10、Q11、Q12、Q14 统计一次，A / B / C / D 哪个最多，就得到第 3 位结果码。
- Q1、Q6、Q13、Q16 统计一次，A / B / C / D 哪个最多，就得到第 4 位结果码。
- 四位结果码合并后，匹配对应主队。

### 平票规则
- 如果某个维度出现平票，优先选择用户在该维度中最早出现的那个选项。

### 示例
- 如果用户在 Q5、Q8、Q9、Q15 中 A 最多
- Q2、Q3、Q4、Q7 中 C 最多
- Q10、Q11、Q12、Q14 中 D 最多
- Q1、Q6、Q13、Q16 中 B 最多

那么他的结果码就是：ACDB。
